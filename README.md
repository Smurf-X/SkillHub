# SkillHub

OpenCode Skills Collection - 自定义技能集合

## Skills

### generate-manual

用于为项目生成手册文档的技能，支持：
- 交互式章节选择
- Todo 进度跟踪
- 分章节输出 Markdown 文件

**使用方式：** 在 OpenCode 中说 "使用 generate-manual skill 生成手册"

## 安装方法

将 `skills/` 目录下的技能文件夹复制到 OpenCode 技能目录：

```
Windows: C:\Users\<用户名>\.config\opencode\skills\
Linux/Mac: ~/.config/opencode/skills/
```

## 目录结构

```
SkillHub/
├── README.md
├── skills/
│   └── generate-manual/
│       └── SKILL.md
└── 手册目录和要求.txt
```
